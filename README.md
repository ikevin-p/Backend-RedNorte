# RedNorte - Plataforma Inteligente de Gestión de Listas de Espera

RedNorte es una plataforma digital orientada a mejorar la gestión de listas de espera en el sistema público de salud.
Buscamos optimizar la asignación de horas médicas, reducir tiempos de espera y mejorar la comunicación con los pacientes.



# Objetivo

Desarrollar una solución tecnológica que permita:

* Gestionar listas de espera de pacientes.
* Optimizar la reasignación de citas médicas canceladas.
* Mejorar la visibilidad del estado de atención para los pacientes.
* Facilitar la toma de decisiones mediante información centralizada.

# Arquitectura General

Basada en microservicios, incluyendo:

* API Gateway / BFF
* Microservicio de Lista de Espera.
* Microservicio de Agenda y Reasignación.
* Microservicio de Notificaciones.

Cada microservicio es independiente y posee su propia base de datos.
